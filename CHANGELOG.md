# Changelog

All notable changes to the dataset are documented here.

Format: `[version] YYYY-MM-DD — summary`

---

## [1.0.0] 2025-03-14 — Initial public release

- 6,145 events across 28 JSON files (1973–2026)
- 183 unique cities
- 3,944 events with GPS coordinates
- 1,371 events with rich Content data
- Earliest event: Nihongo Nihon Bunkasai, Unpad Bandung, 1973-05-23
- Complete Yamato Damashii series documented (Ed. I–XVIII, 2006–2025)
- Gelar Jepang UI series documented (Ed. 1–29, 1995–2023)
- Bunkasai UNPAD / Fesbukan series documented (2001–2023)

---

## Versioning convention

`MAJOR.MINOR.PATCH`

- **MAJOR**: Structural changes to the schema or significant dataset overhaul
- **MINOR**: Addition of 50+ new events or a new year file
- **PATCH**: Corrections, Content field additions, link updates
